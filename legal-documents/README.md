# Documents Légaux - Templates Réutilisables

Ce dossier contient les templates réutilisables pour les documents légaux de votre application.

## Fichiers inclus

- **terms-of-use.tsx** - Conditions d'utilisation (Terms of Use)
- **privacy-policy.tsx** - Politique de confidentialité (Privacy Policy)
- **legal-notice.tsx** - Mentions légales (Legal Notice)

## Utilisation

Ces fichiers peuvent être copiés et adaptés pour d'autres projets. Assurez-vous de :

1. Remplacer les informations spécifiques au projet (nom du site, adresse, email, etc.)
2. Adapter la description du service selon votre projet
3. Vérifier que les informations légales sont à jour
4. Personnaliser le style si nécessaire (couleurs, design)

## Notes

- Les fichiers sont des composants React/Next.js avec Tailwind CSS
- Ils utilisent le même style visuel (gradient slate/emerald)
- Les liens de retour vers la page d'accueil utilisent `href="/"` - adaptez selon votre structure de routes
